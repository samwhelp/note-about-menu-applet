

# demo-mouse-button-modifier


## Usage


### default

run

``` sh
./mouse-button-modifier.sh
```

or run

``` sh
THE_DEFAULT_MENU_FACTORY=fzf THE_DEFAULT_DE=mate ./mouse-button-modifier.sh
```

### alter

run

``` sh
THE_DEFAULT_MENU_FACTORY=rofi THE_DEFAULT_DE=gnome ./mouse-button-modifier.sh
```
