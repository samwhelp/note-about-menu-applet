

# demo-style-select


## Usage


### default

run

``` sh
./style-select.sh
```

or run

``` sh
THE_DEFAULT_MENU_FACTORY=fzf THE_DEFAULT_DE=mate ./style-select.sh
```

### alter

run

``` sh
THE_DEFAULT_MENU_FACTORY=rofi THE_DEFAULT_DE=gnome ./style-select.sh
```
