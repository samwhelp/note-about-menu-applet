## Usage / Menu Factory


### fzf

run

``` sh
./demo.sh
```

or run

``` sh
THE_DEFAULT_MENU_FACTORY=fzf ./demo.sh
```

### dmenu

run

``` sh
THE_DEFAULT_MENU_FACTORY=dmenu ./demo.sh
```

### rofi

run

``` sh
THE_DEFAULT_MENU_FACTORY=rofi ./demo.sh
```

### jgmenu

run

``` sh
THE_DEFAULT_MENU_FACTORY=jgmenu ./demo.sh
```
