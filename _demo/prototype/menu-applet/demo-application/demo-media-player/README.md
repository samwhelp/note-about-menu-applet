

# demo-media-player


## Usage


### default

run

``` sh
./media-player.sh
```

or run

``` sh
THE_DEFAULT_MENU_FACTORY=fzf THE_DEFAULT_PLAYER=mpv ./media-player.sh
```

### alter

run

``` sh
THE_DEFAULT_MENU_FACTORY=rofi THE_DEFAULT_PLAYER=vlc ./media-player.sh
```
