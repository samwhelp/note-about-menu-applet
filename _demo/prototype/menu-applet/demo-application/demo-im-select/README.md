

# demo-im-select


## Usage


### default

run

``` sh
./im-select.sh
```

or run

``` sh
THE_DEFAULT_MENU_FACTORY=fzf THE_DEFAULT_DISTRO=ubuntu ./im-select.sh
```

### alter

run

``` sh
THE_DEFAULT_MENU_FACTORY=rofi THE_DEFAULT_DISTRO=arch ./im-select.sh
```


## Usage / Distro

### ubuntu
