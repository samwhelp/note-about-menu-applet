

# demo-wallpaper-select


## Usage


### default

run

``` sh
./wallpaper-select.sh
```

or run

``` sh
THE_DEFAULT_MENU_FACTORY=fzf THE_DEFAULT_DE=mate ./wallpaper-select.sh
```

### alter

run

``` sh
THE_DEFAULT_MENU_FACTORY=rofi THE_DEFAULT_DE=gnome ./wallpaper-select.sh
```
