

# demo-style-select


## Usage


### mate

run

``` sh
./style-select.sh
```

or run

``` sh
THE_DEFAULT_DE=mate ./style-select.sh
```


### gnome

``` sh
THE_DEFAULT_DE=gnome ./style-select.sh
```


### cinnamon

``` sh
THE_DEFAULT_DE=cinnamon ./style-select.sh
```


### xfce

``` sh
THE_DEFAULT_DE=xfce ./style-select.sh
```
